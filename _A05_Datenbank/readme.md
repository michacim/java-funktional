# Aufgabe Datenbank
* einfache Lösung ohne besondere Struktur
1)

    Erzeuge über PHPMyAdmin eine Datenbank ihk_persons und eine Tabelle persons.
    Die Tabelle soll die folgenden Felder haben:
    id, fistname, lastname,  birthdate.

            Wähle geeignete Datentypen (und für firstname, lastname Größen) aus!

2)

    Erzeuge ein Java-Projekt: _A05_Datenbank.

            Schreibe in der main für diese Datenbank:
    •	Connection
    •	Daten einfügen
    •	Alle Daten anzeigen
    •	Löschen per Id
